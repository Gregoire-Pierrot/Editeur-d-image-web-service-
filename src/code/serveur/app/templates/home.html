Projet web service : Modification d'image

    Bonjour et bienvenue sur notre web service de modification d'images.
    C'est un projet proposé par monsieur SALAS Daniel, responsable de l'UCE Processus du développement logiciel du M1 ILSEN à l'université Avignon.

    Pour pouvoir utiliser ce service vous devez au préalable vous créer un compte. Une fois celui fait vous pourez soit directement utiliser le service avec votre navigateur sur la page : service ou bien alors utilisé l'application disponible sur notre github. Pour pouvoir utiliser cette application vous devez générer un token sur la page : token. Attention celui ne sera pas fournit à nouveau, vous devez donc le concerver. Une fois cela fait il ne vous restera plus qu'a lancer l'application et y rentré votre token. Vous pourez ensuite modifier vos images comme vous le souhaitez dans la mesure de l'utilisation possible.
